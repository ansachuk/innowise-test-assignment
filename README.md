## How to Run the Project

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

## Build

To build the application for production:

```bash
npm run build
```

## Preview

To preview the built application:

```bash
npm run preview
```

## Project Structure

### `/src` - Source Code

Contains all application source files organized by type and purpose.

#### `/src/assets` - Static Assets

Stores static assets like icons.

#### `/src/js` - JavaScript Modules

Contains all JavaScript application logic and functionality.

- **File types**: ES6 modules (.js)

#### `/src/js/utils` - Utility Functions

Contains reusable utility functions and helper modules.

- **File types**: Utility modules (.js)

#### `/src/styles` - CSS Stylesheets

Contains all styling files organized by scope and purpose.

- **File types**: CSS files (.css)

#### `/src/styles/utils` - Utility Styles

Contains CSS files with utility classes and global styles.

- **File types**: Utility CSS files (.css)

#### `/src/styles/components` - Component Styles

Contains CSS files specific to individual UI components.

- **File types**: Component-specific CSS files (.css)
